## 📋 Modules disponibles
- 🏠 **ACCUEIL** — Tableau de bord complet (CA, Bénéfices, Dettes, Alertes)
- 📦 **ARTICLES** — Gestion du stock avec alertes stock bas
- 💰 **VENTE** — Saisie ventes avec calcul bénéfice auto + facture PDF
- 🛒 **ACHAT/IMPORT** — Suivi seuil BEA 30j + coût réel vs officiel
- 💳 **CAISSE** — Entrées/sorties avec solde en temps réel
- 🔴 **DETTES CLIENTS** — Suivi créances + alertes retard
- 🟠 **DETTES FOURNISSEURS** — Suivi en EUR + équivalent DZD
- 📋 **CHARGES** — Loyer, transport, douane, salaires...
- 🚚 **SUIVI COLIS** — Tracker 5 étapes + bon de livraison PDF
- ⚖️ **AJUSTE STOCK** — Inventaire et corrections
- 📊 **STATISTIQUES** — Graphiques + top articles
- ⚙️ **PARAMÈTRES** — Infos société, QR Code factures

## 📱 Installation
**PC:** Icône dans la barre d'adresse Edge/Chrome → Install
**Android:** Chrome → Menu → "Ajouter à l'écran d'accueil"
**Mise à jour:** Automatique à chaque ouverture

## 🚀 Déploiement GitHub
1. Créer repo `gestion-pro` sur github.com
2. Uploader les 5 fichiers
3. Settings → Pages → main → Save
